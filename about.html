Projects

VLM UI: Vision Language Model User Interface

VLM UI is a pioneering web-based user interface designed for efficient interaction with Vision Language Models (VLMs). Unlike existing tools for Large Language Models (LLMs), VLM UI fills a gap in the market by providing:

  • User-friendly frontend for multiple VLMs
  • Current support for PaliGemma, Phi3-Vision, and Florence-2
  • Plans for integration with additional VLMs
  • Complementary Command Line Interface (CLI) for task automation

This project evolved from separate UIs for PaliGemma and Phi3-Vision, which are available on GitHub. VLM UI is a personal project that I'm developing independently in my free time, aimed at enhancing accessibility and usability in the growing field of vision-language AI.

DVS 3: Student Union Printing System Management Tool

DVS 3 is a comprehensive web-based management tool for our university's student union printing system. This project represents a complete rewrite of the previous PHP-based DVS 2 system, with significant enhancements:

  • Developed using Python with the Django framework
  • React-based frontend for improved user experience
  • New functionalities: e.g. LDAP integration and enhanced inventory management
  • Collaborative effort with three other students

My primary contributions to this team project include:

  • Payment processing, including state flow management
  • Authentication system implementation (frontend and backend)
  • LaTeX-based document generation and email notification

This project increased my proficiency in full-stack web development, team collaboration, and creating practical solutions for real-world organizational needs.

Connected Lamps Project

Inspired by the K-drama "I'm Not a Robot," I developed a pair of internet-connected lamps. Key features include:

  • ESP32 microcontroller-based, programmed in C++ using PlatformIO
  • Dual control options: physical buttons or custom Android app written in Kotlin
  • Bluetooth Low Energy (BLE) connectivity between app and ESP32s
  • Original ESP-BLE interface code published on GitHub as an open-source template
  • Lamps interconnected via WebSocket server
This project helped me develop my skills in IoT development, mobile app creation, and even led to open-source contribution.
